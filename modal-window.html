Escaping the Modal

With a modal you are mimicking an operating system dialog box. Thus there is no way to leave the dialog without either:

  • engaging the actionable items in the dialog box, or
  • hitting the ESC key.

A modal window should mimic this behavior. As long as the user has a way out either by using the close button or by hitting “ESC” the modal is fine.
