iFrames are difficult to hide from screen readers. Just using display: none will often not keep screen readers from caching and reading the content inside. (https://www.paciellogroup.com/blog/2010/04/making-sure-hidden-frames-are-hidden/)

Back in 2010 Steve Faulkner mentioned someday being able to use the HTML5 global hidden attribute. This attribute is now fully supported.
