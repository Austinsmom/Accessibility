The list below highlights a list of tools that can be used by folks with disorders (the format is as follows: Tool - Platform - Use of Tool):

NVDA - FireFox Browser / Windows - Screenreader for users with visual or reading impairment

VoiceOver - Apple Devices - Screenreader for users with visual or reading impairment

JAWS - Internet Explorer Browser / Windows - Screenreader for users with visual or reading impairment

Dragon NaturallySpeaking - Windows / Macintosh - Speech recognition tool for users with visual impairment, motor and cognitive disabilities

Amazon Rekognition - All Devices / On Web - Visual recognition of objects and faces for people with cognitive disabilities

Tap Tap See - iOS Application - Visual recognition of objects for people with visual impairments

Be My Eyes - iOS Application / iPhone Devices - For People with visual impairments, the app connects people who need assistance with sighted volunteers

Google Translate - All Devices - Allows reading of physical paper for users with visual impairments

TalkBack - Android Devices - Screenreader for users with visual or reading impairment
