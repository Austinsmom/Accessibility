Is aria-hidden recognized when using the up and down arrows?

Sample Text

This is a paragraph.

This is a fake link.

Now this paragraph is supposed to be hidden from screen-readers by using aria-hidden="true"

This is another regular paragraph (not hidden from screen readers).

The Test:

  • Open page in browser with screen reader on.
  • Have screen reader read the entire page.
  • Next, use the up and down arrows to review the page.

Expected result: The aria-hidden paragraph should not be read.

NOTE: This experiment does not take into account Quick Nav or Trackpad Commander in Safari on Mac. It simply tests whether the user can access aria-hidden content using up and down arrows.

My Findings
Was aria-hidden content read: JAWS with IE NVDA with Firefox VoiceOver on Safari (Mac) VoiceOver on Safari (iPad) VoiceOver on Safari (iPhone) TalkBack on Chrome (Galaxy Tablet) TalkBack on Chrome (Galaxy Phone)
when page is automatically read No No No
when up and down arrows are used No No Yes

NOTE: When using Quick Nav or Trackpad Commander in Safari on Mac aria-hidden="true" is honored.
